# CATCH App 📱

## 📌 About the Project
CATCH is a location-based social mobile app that allows users to discover nearby people based on real-time location.

## 🚀 Features
- User authentication (Firebase)
- Real-time location tracking
- Nearby users list
- Filtering by age and gender
- User profile with social links
- Profile image upload (Cloudinary)
- Bottom navigation UI

## 🛠️ Technologies
- Android (Java)
- Firebase Authentication
- Firestore Database
- Cloudinary
- Google Location Services
